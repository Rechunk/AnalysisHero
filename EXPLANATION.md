## 1. Calculating the derivation

Here is an example on how the derivation is calculated:

## 1. Function: 3x^2 + 5x
## 2. List:    [3x^2, +, 5x]
## 3. RegEx:   ([3 = factor, x is there, 2 = exponent], +, [x is there, 5 = factor])
## 4. Rule no.1: ([factor * exponent, exponent - 1], +, [remove x])
